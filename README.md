# LoRaWAN байланысы арқылы өнеркәсіптегі датчиктерді интеллектуалды басқару жүйесі

##  Жобаның сипаттамасы
Бұл жоба өнеркәсіптік ортадағы датчиктерді LoRaWAN сымсыз байланыс технологиясы арқылы интеллектуалды басқаруға арналған IoT жүйе болып табылады.  

Жүйе датчиктерден (температура, ылғалдылық, қысым және т.б.) деректерді жинап, оларды орталық серверде өңдейді және алынған ақпарат негізінде өндірістік құрылғыларды (насос, клапан, желдеткіш және т.б.) автоматты түрде басқарады.

Жобаның негізгі мақсаты – өнеркәсіптік процестерді автоматтандыру, ресурстарды тиімді пайдалану және адам факторын азайту.

---

##  Жүйенің негізгі мүмкіндіктері
-  LoRaWAN арқылы сымсыз байланыс
-  Датчиктерден нақты уақыт режимінде дерек жинау
-  Интеллектуалды шешім қабылдау жүйесі
-  Құрылғыларды автоматты басқару
-  Деректерді өңдеу және мониторинг
-  Авариялық ескерту және хабарламалар жүйесі
-  Масштабталатын IoT архитектура

---

##  Жүйе архитектурасы
Жүйе келесі логикалық құрылым бойынша жұмыс істейді:

**Датчик → LoRaWAN → Gateway → Сервер → Өңдеу → Шешім қабылдау → Атқару → Кері байланыс**

Бұл құрылым үздіксіз бақылау және автоматты басқаруды қамтамасыз етеді.

---

##  Қолданылған технологиялар
- LoRaWAN байланыс протоколы
- IoT датчиктер жүйесі
- Python / C++ (жобаның бөлігіне байланысты)
- FastAPI / Backend сервер
- PostgreSQL / SQLite деректер қоры
- Replit / Wokwi / Tinkercad симуляция құралдары

---

##  Жұмыс логикасы
1. Датчиктер орта параметрлерін өлшейді  
2. Деректер LoRaWAN арқылы жіберіледі  
3. Сервер ақпаратты қабылдап, өңдейді  
4. Алгоритм жағдайды талдайды  
5. Жүйе шешім қабылдайды  
6. Құрылғыларға басқару сигналдары жіберіледі
